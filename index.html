I am full stack developer whose been developing for some years now. Got my first crack at web developing in middle school with basic HTML which eventually evolved into Full Stack development after graduating CityTech college. Furthered my education with Udacity's Full Stack Nanodegree which lead to me wanting to learn more. Now, I'm developing front end applications with React/Redux as well as Full Stack applications using Python/Django/Flask, HTML/CSS, Javascript, JQuery.

I've always had a curious mind of how things work so Full stack web development fulfills the need of understanding how web application with how data flows on the backend from the front end as well my artistic of crafting beautiful interfaces. When I am not painting with IDE as my canvas using code, Im most likely playing basketball, learning new skateboard tricks, or adventuring around Queens.
