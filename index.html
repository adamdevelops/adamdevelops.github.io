Learn about my journey as a developer.

I am full stack developer whose been developing for some years now. I first got caught with the coding bug since the first time I learned HTML in high school. My first website has to be one of my favorite creations. To be honest, my first site is outdated by todays standards. I wouldn't have of dreamed doing half styling/animations things on this portfolio site with the knowledge I had back then. Ever since then I been amazed at what can be created on a web page as my canvas.

With my love for computers, I studied at NewYork City College Of Technology. This where I especially honed my skill in programming and problem solving. My experience there really humbled me and I discovered my knack for programming. This is where my eyes where open to the wonders of hybrid web mobile development with my work on a programming quiz app for one of CityTechs classes. After this, my passion for web development was ignited. I must learn more about web development.

I would learn that there is more to websites than just the way they look, the Front End. I found out there is BackEnd development where you can also control the data circulated, store it, and display it. You're telling me, I can control the way my site looks and how data is stored, sign me up. So I would continue my education and add more tinder to the fire by enrolling in Udacitys Full Stack Development Nanodegree. Learning with Udacity, I build my foundational knowledge for both Front End and Back End development. During this nanodegree I built many projects such as a Neighborhood Map of spots to go to in Chinatown, full fledged blog that any user can log in and post content, and hosting my very own web server that hold one my projects that I created. Since the first day I learned about web development and caught the coding bug and I have been debugging it ever since.
